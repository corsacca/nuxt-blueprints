# Kitchen Sink Block

A UI component showcase page for development reference. Demonstrates Nuxt UI components, theming, icons, form elements, and layout patterns.

## What You Get

A single page at `/kitchen` that displays all available Nuxt UI components with various configurations, so you can see how they look with your theme and copy patterns into your own pages.

## Files

```
app/pages/kitchen.vue
```

## Environment Variables

None.

## Dependencies

- `core` (Nuxt UI, theming)

## npm Packages

None beyond what `core` provides.
